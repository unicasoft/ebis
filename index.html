⚠ Kamera için HTTPS Gerekli
Chrome/Firefox güvenlik politikası gereği kamera yalnızca HTTPS veya localhost üzerinden çalışır.
Resimden okuma ve metin girişi her zaman çalışır. Kamera için:

Python
Node.js
Nginx/Apache
Bu HTML dosyasının bulunduğu klasörde terminal açıp:
python3 -m http.server 8080
Sonra tarayıcıda açın: http://localhost:8080/ebis-karekod-dogrulayici-v2.html
npx serve .
Sonra: http://localhost:3000/ebis-karekod-dogrulayici-v2.html
Dosyayı sunucunuzun web klasörüne kopyalayın.
SSL sertifikası varsa (Let's Encrypt) otomatik çalışır.
Ya da GitHub Pages / Netlify'a yükleyin — ücretsiz HTTPS verir.
